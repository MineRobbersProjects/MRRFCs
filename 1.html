ComputerCraft File Transfer Protocol - CCFTP


1. Introduction

CCFTP is a program/protocol that allows files to be transferred from one capable device to another capable device. A "capable device", in this context, is an in-game device capable of storing files.

2. Connection

In order to establish a connection with a "server", the client must send it a "ccftp:connect" message. This message will let the server know that the client wishes to connect.
